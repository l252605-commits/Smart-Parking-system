# Smart Parking System 🚗

## Overview

The Smart Parking System is a console-based application developed using Object-Oriented Programming (OOP) concepts. The system helps manage parking operations efficiently by handling vehicle entry, exit, parking slot allocation, fee calculation, and record management.

## Features

* Vehicle Entry Management
* Vehicle Exit Management
* Automatic Parking Slot Allocation
* Parking Fee Calculation
* Available Slot Tracking
* Vehicle Record Management
* User-Friendly Menu Interface
* File Handling for Data Storage

## OOP Concepts Implemented

* Classes and Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Constructors and Destructors

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling

## System Functionalities

1. Add a vehicle to the parking lot.
2. Assign an available parking slot.
3. Display parking status and available slots.
4. Calculate parking charges based on parking duration.
5. Remove vehicles upon exit.
6. Store and retrieve parking records.
